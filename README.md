## 개발 검증

`node tools/verify-runtime.mjs`는 Foundry를 실행하거나 월드·컴펜디움·팩을 수정하지 않는 읽기 전용 검사입니다. 매니페스트/로케일 JSON, 선언한 스크립트·스타일 경로, 런타임 JavaScript 구문, 코드에 리터럴로 적은 `DX3rd.*` i18n 키를 확인합니다. 실제 판정·소켓·전투 흐름은 격리된 검증 월드에서 수동으로 확인합니다.
